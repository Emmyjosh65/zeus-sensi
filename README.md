#  Free Fire Sensitivity AI

AI-powered Free Fire sensitivity calculator for:
- Smooth aim
- Zero recoil
- Fast headshots
- Better drag accuracy

---

## Features

- AI sensitivity generation
- Device-based optimization
- FPS adjustment
- DPI recommendations
- Headshot presets
- Supports all Android devices

---

## Supported Settings

- General
- Red Dot
- 2X Scope
- 4X Scope
- Sniper Scope
- Free Look
